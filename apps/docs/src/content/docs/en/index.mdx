<div class="trc-features">
  <div class="trc-feature">
    <div class="trc-feature-icon">&#9776;</div>
    <h3>4 Views</h3>
    <p>Month, Week, Day, and Agenda — each with polished, responsive rendering and overlap handling.</p>
  </div>
  <div class="trc-feature">
    <div class="trc-feature-icon">&#8693;</div>
    <h3>Drag & Drop</h3>
    <p>Native HTML5 DnD. Move events across days and time slots. 15-minute snap. Zero extra dependencies.</p>
  </div>
  <div class="trc-feature">
    <div class="trc-feature-icon">&#127760;</div>
    <h3>Full i18n</h3>
    <p>Native Intl API for dates. Translatable UI labels. No moment.js, no date-fns. Just the platform.</p>
  </div>
  <div class="trc-feature">
    <div class="trc-feature-icon">&#9881;</div>
    <h3>Slots API</h3>
    <p>Replace any sub-component — toolbar, event pills, day cells, popovers — with your own React components.</p>
  </div>
  <div class="trc-feature">
    <div class="trc-feature-icon">&#9790;</div>
    <h3>Theming</h3>
    <p>CSS variables with shadcn/ui fallback chain. Dark mode built in. Zero-config when paired with shadcn.</p>
  </div>
  <div class="trc-feature">
    <div class="trc-feature-icon">&#9881;</div>
    <h3>Headless Core</h3>
    <p>trud-calendar-core: types, date math, layout algorithms, state — zero deps, any framework.</p>
  </div>
</div>
