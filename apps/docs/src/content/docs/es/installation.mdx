## Instalar el paquete

```bash
npm install trud-calendar
# or
pnpm add trud-calendar
# or
yarn add trud-calendar
```

**Peer dependencies:** `react >=18`, `react-dom >=18`

## Configuracion con Tailwind v4

Agrega la fuente del paquete a tu CSS para que Tailwind genere las clases de utilidad necesarias:

```css
@import "tailwindcss";
@source "../node_modules/trud-calendar/dist";
```

Importa las variables de tema (opcional si ya usas shadcn — las variables se heredan automaticamente):

```css
@import "trud-calendar/styles.css";
```

## Sin Tailwind

Si no usas Tailwind, simplemente importa el CSS pre-construido:

```css
@import "trud-calendar/styles.css";
```

O en tu archivo JavaScript de entrada:

```ts
import "trud-calendar/styles.css";
```

## Proyectos con shadcn/ui

Si tu proyecto ya usa shadcn/ui, el calendario hereda tu tema automaticamente. Las variables CSS hacen fallback a las de shadcn: `var(--background)`, `var(--foreground)`, etc. No necesitas configuracion extra.

Solo asegurate de agregar la directiva `@source` en tu CSS para que Tailwind genere las clases de utilidad del calendario.
